# ioBroker Sigenergy Адаптер

[![NPM version](https://img.shields.io/npm/v/iobroker.sigenergy.svg)](https://www.npmjs.com/package/iobroker.sigenergy)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Адаптер для солнечных энергетических систем Sigenergy через Modbus TCP/RTU**

Поддерживает протокол Sigenergy Modbus V2.5 (выпущен 2025-02-19).

---

## Функции

- 📡 **Modbus TCP** (Ethernet / WLAN / оптоволокно / 4G) — порт 502
- 🔗 **Modbus RTU** (RS485 последовательный)
- ⚡ **Полная поддержка регистров** — все регистры установки и инвертора по спецификации V2.5
- 🔋 **Статистика батареи** — время до полного заряда, оставшееся время, суточное покрытие
- ☀️ **Статистика PV** — доля собственного потребления, степень автономности
- 🔌 **AC-зарядное устройство** (Sigen EVAC) — опционально
- ⚡ **DC-зарядное устройство** — опционально
- 📊 **Вычисляемые значения** — производные статистики обновляются при каждом цикле опроса
- 🖥️ **VIS-виджеты** — поток энергии, состояние батареи, панели статистики

## Поддерживаемое оборудование

| Категория        | Модели |
|-----------------|--------|
| **Гибридный инв.** | SigenStor EC SP/TP, Sigen Hybrid SP/TP/TPLV, Sigen PV M1-HYA, PG Controller |
| **PV инв.**     | Sigen PV Max SP/TP, Sigen PV M1 |
| **EVAC (AC)**   | Sigen EVAC 7/11/22 кВт, PG EVAC |

| Устройство | Адрес |
|-----------|-------|
| Установка (чтение/запись) | **247** |
| Широковещание установки (запись, без ответа) | **0** |

| AC-зарядное устройство (EVAC) | **2** |

## Установка

### Через ioBroker Admin (рекомендуется)
1. Открыть ioBroker Admin → Адаптеры
2. Найти «sigenergy»
3. Установить

## Конфигурация

### Вкладка Подключение
- **Тип подключения**: TCP (Ethernet) или Последовательный (RS485)
- **TCP-хост**: IP-адрес инвертора
- **TCP-порт**: 502 (по умолчанию)
- **Modbus ID установки**: 247 (по умолчанию)
- **Modbus ID инвертора**: 1 (по умолчанию)

### Вкладка Компоненты
Выбор установленных устройств:
- Батарея / ESS
- Солнечные панели
- AC-зарядное устройство (EVAC)
- DC-зарядное устройство

### Вкладка Статистика
Выбор вычисляемых статистических значений:
- Время до полного заряда батареи
- Оставшееся время работы батареи
- Ежедневное время зарядки
- Время покрытия батареей
- Доля собственного потребления
- Степень автономности

## Объекты данных

### Установка (`plant.*`)
| Состояние | Описание | Единица |
|-----------|---------|---------|
| `plant.gridActivePower` | Мощность сети (>0 потребление, <0 экспорт) | кВт |
| `plant.pvPower` | Выработка PV | кВт |
| `plant.essPower` | Мощность батареи (<0 разряд) | кВт |
| `plant.essSoc` | Заряд батареи | % |
| `plant.activePower` | Суммарная активная мощность установки | кВт |
| `plant.runningState` | Состояние установки (0=Ожидание, 1=Работа...) | - |

### Инвертор (`inverter.*`)
| Состояние | Описание | Единица |
|-----------|---------|---------|
| `inverter.pvPower` | Мощность PV на инверторе | кВт |
| `inverter.essBatterySoc` | SOC батареи | % |
| `inverter.essBatterySoh` | SOH батареи | % |
| `inverter.essBatteryTemperature` | Температура батареи | °C |
| `inverter.phaseAVoltage` | Напряжение фазы A | В |
| `inverter.gridFrequency` | Частота сети | Гц |

### Статистика (`statistics.*`)
| Состояние | Описание | Единица |
|-----------|---------|---------|
| `statistics.batteryTimeToFull` | Минут до полного заряда батареи | мин |
| `statistics.batteryTimeRemaining` | Оставшееся время батареи | мин |
| `statistics.selfConsumptionRate` | Доля собственного потребления | % |
| `statistics.autarkyRate` | Степень автономности | % |
| `statistics.housePower` | Вычисленное потребление дома | кВт |

## VIS-виджеты

> **Примечание:** Все 7 виджетов предоставляются отдельным адаптером [ioBroker.vis-2-widgets-sigenergy](https://github.com/ssbingo/ioBroker.vis-2-widgets-sigenergy). Установите его вместе с этим адаптером для использования виджетов в VIS-2.

### Виджет потока энергии
Показывает анимированный поток энергии PV → Батарея ↔ Сеть → Дом.

### Виджет состояния батареи
Отображает индикатор SOC, значок SOH, время до полного заряда/разряда, текущую мощность.

### Виджет обзора мощности
Живые показания всех четырёх потоков мощности.

### Виджет статистики
Сегодняшняя автономность, самопотребление, мин/макс SOC, время покрытия батареей.

### Виджет инвертора
Данные инвертора в реальном времени: мощность PV, частота сети, напряжения фаз, температура.

### Виджет AC-зарядного устройства (EVAC)
Статус и показания мощности зарядной станции Sigen EVAC.

### Виджет DC-зарядного устройства
Статус и показания мощности DC-зарядного устройства.

---

## Протокол связи

- Modbus TCP: режим TCP, полный дуплекс, порт 502 (ведомый)
- Modbus RTU: полудуплекс, 9600 бит/с, 8N1
- Минимальный интервал опроса: 1000 мс (1 секунда) по спецификации Sigenergy
- Тайм-аут: 1000 мс по спецификации Sigenergy
