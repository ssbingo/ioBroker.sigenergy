## Функції

- 📡 **Modbus TCP** (Ethernet / Wi-Fi / Оптоволокно / 4G) — Порт 502
- 🔗 **Modbus RTU** (RS485 серійний)
- ⚡ **Повна підтримка регістрів** — Усі регістри установки, інвертора, PSS та PID згідно зі специфікацією V2.9
- 🔋 **Статистика акумулятора** — Час до повного заряду, залишковий час, денне покриття
- ☀️ **Статистика ФЕС** — Рівень власного споживання, рівень автономності
- 🔌 **AC-зарядний пристрій** (Sigen EVAC) — Необов'язковий
- ⚡ **DC-зарядний пристрій** — Необов'язковий
- 🏗️ **PSS** (силовий перемикач) — Опціонально, моніторинг СН/НН розподільних пристроїв
- 🔍 **PID** (контроль ізоляції PV) — Опціонально
- 🌡️ **Попереднє нагрівання ESS** — Розклад TOU, 30 налаштовуваних часових вікон (M1-HYA/HYB)
- 📈 **Розширені регістри** — Розумні навантаження 1–24, лічильники енергії, параметри кодексу мережі
- ☀️ **SigenMicro** — Підтримка мікроінверторів (автосканування)
- 📊 **Розраховані значення** — Похідна статистика оновлюється з кожним циклом опитування
- 🖥️ **VIS-віджети** — Потік енергії, стан акумулятора, панелі статистики

## Конфігурація

### Вкладка «Підключення»
- **Тип з'єднання**: TCP (Ethernet) або Серійний (RS485)
- **TCP-хост**: IP-адреса інвертора
- **TCP-порт**: 502 (за замовчуванням)
- **Modbus ID установки**: 247 (за замовчуванням)
- **Modbus ID інвертора**: 1 (за замовчуванням)

### Вкладка «Компоненти»
Виберіть встановлені пристрої:
- Акумулятор / ESS
- ФЕС-панелі
- AC-зарядний пристрій (EVAC)
- DC-зарядний пристрій
- PSS (силовий перемикач)
- PID (контроль ізоляції PV)
- Попереднє нагрівання ESS (тільки M1-HYA/HYB)
- SigenMicro (мікроінвертори)

### Вкладка «Статистика»
Виберіть розраховані статистичні значення:
- Час до повного заряду акумулятора
- Залишковий час акумулятора
- Щоденний час заряду
- Час покриття акумулятора
- Рівень власного споживання
- Рівень автономності

## Об'єкти даних

### Установка (`plant.*`)
| Стан | Опис | Одиниця |
|------|------|---------|
| `plant.gridActivePower` | Потужність мережі (>0 споживання, <0 подача) | кВт |
| `plant.pvPower` | Вироблення ФЕС | кВт |
| `plant.essPower` | Потужність акумулятора (<0 розряд) | кВт |
| `plant.essSoc` | Рівень заряду акумулятора | % |
| `plant.activePower` | Загальна активна потужність установки | кВт |
| `plant.runningState` | Стан установки (0=Очікування, 1=Робота...) | - |

### Інвертор (`inverter.*`)
| Стан | Опис | Одиниця |
|------|------|---------|
| `inverter.pvPower` | Потужність ФЕС на інверторі | кВт |
| `inverter.essBatterySoc` | SOC акумулятора | % |
| `inverter.essBatterySoh` | SOH акумулятора | % |
| `inverter.essBatteryTemperature` | Температура акумулятора | °C |
| `inverter.phaseAVoltage` | Напруга фази A | В |
| `inverter.gridFrequency` | Частота мережі | Гц |

### Статистика (`statistics.*`)
| Стан | Опис | Одиниця |
|------|------|---------|
| `statistics.batteryTimeToFull` | Хвилин до повного заряду | хв |
| `statistics.batteryTimeRemaining` | Залишковий час акумулятора | хв |
| `statistics.selfConsumptionRate` | Рівень власного споживання | % |
| `statistics.autarkyRate` | Рівень автономності | % |
| `statistics.housePower` | Розрахунковий споживання будинку | кВт |

## VIS-віджети

> **Примітка:** Усі 7 віджетів надаються окремим адаптером [ioBroker.vis-2-widgets-sigenergy](https://github.com/ssbingo/ioBroker.vis-2-widgets-sigenergy). Встановіть його разом із цим адаптером для використання віджетів у VIS-2.

### Віджет потоку енергії
Відображає анімований потік енергії: ФЕС → Акумулятор ↔ Мережа → Будинок.

### Віджет стану акумулятора
Показує шкалу SOC, значок SOH, час до повного/порожнього, поточну потужність.

### Віджет огляду потужності
Відображення в реальному часі всіх чотирьох потоків потужності.

### Статистичний віджет
Автономність та власне споживання за сьогодні, SOC мін/макс, час покриття акумулятора.

### Віджет інвертора
Дані інвертора в реальному часі: потужність ФЕС, частота мережі, напруга фаз, температура.

### Віджет AC-зарядного пристрою (EVAC)
Стан і показники потужності зарядної станції Sigen EVAC.

### Віджет DC-зарядного пристрою
Стан і показники потужності DC-зарядного пристрою.

## Протокол зв'язку

- Modbus TCP: режим TCP, повний дуплекс, порт 502 (ведений)
- Modbus RTU: напівдуплекс, 9600 бод, 8N1
- Мінімальний інтервал опитування: 1000 мс (1 секунда) згідно зі специфікацією Sigenergy
- Тайм-аут: 1000 мс згідно зі специфікацією Sigenergy
